Good Maowning!
==============
A simple java server project that will send an email to subscribers each morning with a cute cat picture.  


Requirements
============
The following are environment requirements and must be accessable on the path to properly use this project:
jre 1.8.0_31
ant 1.9.4

Additional properties files are also required:
email.properties: 
- contains all the properties needed for the source email address that will be sending the "Good Maowning" messages.
- required properties: maower.email, maower.username, maower.password, maower.host, maower.provider
